DESIGN SYSTEM:
- Paleta corporativa:
  - Primary: HSL 206 59% 44% (#2E7BB4) — botones principales
  - Accent: HSL 209 78% 28% (#0F4C81) — azul corporativo profundo
  - Secondary: HSL 195 47% 94% (#E8F4F8) — fondos claros
  - Success: HSL 160 84% 39% (#10B981)
  - Warning: HSL 38 92% 50% (#F59E0B)
  - Destructive: HSL 0 84% 60% (#EF4444)
  - AI: HSL 258 90% 66% (#8B5CF6) — features de IA
  - Background: HSL 220 14% 96% (#F3F4F6)
  - Foreground: HSL 215 28% 17% (#1F2937)
  - Sidebar: tema oscuro #1F2937
- Border radius: 0.625rem (10px)
- Sombras: usar variables CSS (--shadow-sm, --shadow-md, etc.)
- NUNCA usar colores directos en componentes, siempre tokens semánticos

REGLAS DE UI/UX ESTRICTAS:
1. NUNCA usar height: 100vh → usar h-[100dvh] en móviles
2. SIN scroll horizontal → overflow-x-hidden global
3. Touch: touch-action: pan-x pan-y, -webkit-overflow-scrolling: touch
4. Safe areas iOS: safe-area-left, safe-area-right
5. Inputs en móvil: font-size mínimo 16px para evitar zoom iOS
6. Z-index scale: Sidebar(40), POS Modal(60), Diálogos(70), Alertas(80), Popovers(9999)
7. Estilo móvil compacto (.mobile-compact): padding 0.5rem, botones 36px, texto 0.6875rem
8. NO inventar datos mock/demo para producción. Si no hay datos, mostrar estado vacío elegante

ARQUITECTURA DE LAYOUT:
- MainLayout: sidebar fijo izquierda (desktop/tablet) + header fijo arriba + contenido scrollable
- Mobile (≤767px): sin sidebar, MobileNav fija abajo (5 items principales), pt-12 pb-16, padding 12px
- Tablet (768-1023px): sidebar colapsado (70px), padding 16px
- Desktop (≥1024px): sidebar expandible (260px/70px), padding 24px
- Sidebar oscuro con icono Eye como logo, nombre "Óptica Istmeña" y subtítulo "Suite Admin"
- Footer con crédito al developer en desktop

RESPONSIVE:
- Breakpoints via hook useBreakpoint() → 'mobile' | 'tablet' | 'desktop'
- 3 columnas: useBreakpoint, useIsMobile (boolean), useTouchScroll
- Usar useIOSKeyboard hook para evitar zoom en inputs iOS
